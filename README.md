### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/alwazirf/landing-page-finance.git
   ```
2. Install PNPM packages
   ```sh
   pnpm install
   ```
3. Running this App
   ```sh
   pnpm run dev
   ```
